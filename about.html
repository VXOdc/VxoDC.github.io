About Vxo Study OS

My project is called Vxo Study OS and it is a website I made using HTML, CSS, and JavaScript to help me stay organized with my school work.

It has a system where I can manage school tasks so I do not forget deadlines and assignments.

It also includes a math engine that gives random problems for addition, subtraction, and multiplication so I can practice for tests.

I built it using HTML for structure, CSS for design, and JavaScript for the interactive features.

This project is designed to look and feel like a lightweight operating system for students.
