About VxoDC

My project is called VxoDC and it is a website I made using HTML and CSS and JavaScript to help me stay organized with my school work.

It has a special area for project management where I can write down all my different assignments so I do not forget when they are due to my teachers.

There is also a math practice app built into it that gives me different problems to solve so I can get better at math and practice for my tests.

I used HTML for the structure and CSS to change the colors and JavaScript to make the math problems actually work when I click the buttons.

I worked really hard on the code so that it looks good and helps me finish my projects on time without getting overwhelmed.
